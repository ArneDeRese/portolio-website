Over mij

Tijdens mijn opleiding in het graduaat programmeren heb ik geleerd om webpagina's te bouwen met behulp van drie belangrijke technologieën: HTML, CSS en JavaScript. Deze talen zijn de fundamenten van web ontwikkeling en stellen me in staat om interactieve en visueel aantrekkelijke websites te maken.

Ik begon mijn reis met HTML (HyperText Markup Language), de opmaaktaal die de structuur van een webpagina bepaalt. Door HTML-tags te gebruiken, leer ik hoe ik inhoud kan organiseren, zoals tekst, afbeeldingen en formulieren. Het begrijpen van HTML is essentieel voor het creëren van een duidelijke en goed georganiseerde webpagina.

Daarna leerde ik CSS, waarmee ik de visuele uitstraling van een website kan aanpassen. Het stelt me in staat om kleuren, lay-out en lettertypen te bepalen, zodat mijn website er goed uitziet op verschillende apparaten.

JavaScript maakt het mogelijk om mijn websites interactief te maken. Door JavaScript te gebruiken, kan ik dynamische elementen toevoegen, zoals knoppen, formulieren en animaties, die reageren op gebruikersacties. Dit zorgt ervoor dat mijn webpagina's niet alleen statisch zijn, maar echt met de gebruiker communiceren.

Door deze drie technologieën te combineren, kan ik websites creëren die zowel functioneel als visueel aantrekkelijk zijn.
